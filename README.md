# 个人网页

## 项目结构
```
personal/
├── public/              # 静态资源
├── src/
│   ├── components/      # React组件
│   │   ├── Header.jsx
│   │   ├── Footer.jsx
│   │   ├── Hero.jsx
│   │   ├── About.jsx
│   │   ├── Projects.jsx
│   │   └── Contact.jsx
│   ├── styles/          # 样式文件
│   │   └── index.css
│   ├── assets/          # 图片等资源
│   ├── App.jsx          # 主应用组件
│   └── main.jsx         # 入口文件
├── index.html
├── package.json
└── vite.config.js
```

## 使用方法

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```
